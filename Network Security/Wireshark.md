## Wireshark

Wireshark is an open-source network protocol analyzer widely used by security analysts, network engineers, and incident responders to capture and interactively analyze network traffic in real time. It provides deep visibility into network packets, allowing analysts to inspect individual frames, decode protocols, and reconstruct network sessions to identify anomalous or malicious activity.

Wireshark is a foundational tool in network forensics, intrusion analysis, and traffic-based threat hunting, supporting hundreds of protocols and offering powerful filtering capabilities for focused investigation.

### Objectives of Wireshark Analysis

* Capture and inspect network traffic in real time or from saved packet captures
* Identify malicious or anomalous traffic patterns and communication behaviors
* Reconstruct network sessions and extract transferred files or credentials
* Apply capture and display filters to isolate relevant traffic during investigations
* Support network forensics, incident response, and threat hunting activities

### Common Information Analyzed

* Packet headers, payloads, and protocol fields
* Source and destination IP addresses, ports, and protocols
* DNS queries, HTTP requests, and TLS handshakes
* TCP stream reconstruction and session analysis
* Indicators of compromise within network traffic
* Anomalous traffic volumes, beaconing patterns, and lateral movement
