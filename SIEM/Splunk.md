## Splunk

Splunk is a leading security information and event management (SIEM) platform that enables organizations to collect, index, search, and analyze machine-generated data from across their IT infrastructure in real time. It provides powerful search, correlation, alerting, reporting, and dashboard capabilities through its Search Processing Language (SPL), making it a central tool in security operations centers (SOCs) for threat detection, incident response, and compliance monitoring.

Splunk's Boss of the SOC (BOTS) datasets provide realistic, scenario-based CTF-style challenges that allow analysts to develop and sharpen their Splunk investigation skills using real-world attack data in a hands-on environment.

### Objectives of Splunk Analysis

* Collect and index log data from diverse sources across the enterprise
* Search, correlate, and investigate security events using SPL queries
* Build alerts and automated notifications for threat detection use cases
* Develop reports and dashboards for security monitoring and visibility
* Practice threat investigation skills using Boss of the SOC datasets

### Common Information Analyzed

* Indexed events from endpoints, network devices, and applications
* SPL search queries, transforming commands, and statistical functions
* Correlation searches and alert threshold configurations
* Dashboard panels and visualization of security metrics
* Universal Forwarder deployment and data ingestion configurations
* CTF-based investigation scenarios using BOTS datasets
