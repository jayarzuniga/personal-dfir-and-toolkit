## Sysinternals

Sysinternals is a suite of advanced system utilities and technical information tools developed by Mark Russinovich and acquired by Microsoft. It provides deep visibility into Windows internals, enabling security analysts, incident responders, and system administrators to monitor, troubleshoot, and investigate system activity at a granular level. Sysinternals tools are widely used in digital forensics, malware analysis, and endpoint threat hunting operations.

The suite covers a broad range of capabilities including process monitoring, network connection analysis, autoruns persistence inspection, memory dumping, and user session tracking, making it an essential toolkit for Windows-based investigations.

### Objectives of Sysinternals Analysis

* Monitor and investigate running processes, threads, and system activity
* Identify malicious persistence mechanisms and autostart entries
* Analyze network connections and associated processes on an endpoint
* Capture memory dumps for further malware or forensic analysis
* Support incident response, threat hunting, and live system investigations

### Common Information Analyzed

* Running processes, parent-child relationships, and loaded modules
* Autostart locations including registry keys, scheduled tasks, and startup folders
* Network connections and associated process binaries
* Logged-on users and active sessions on local and remote systems
* File system and disk activity at the driver level
* Registry modifications and real-time change monitoring
* System call and event activity captured via Sysmon event logs
